## Question

Definir el dashboard de métricas fijas del MVP + su integración con SQL queries.

Métricas fijas a definir:
- ¿Top 5 productos más vendidos?
- ¿Ventas del día/semana/mes?
- ¿Productos con stock bajo?
- ¿Clientes frecuentes?
- ¿Margen por producto/categoría?
- ¿Cuentas a cobrar / fiados? (si aplica)

Para cada métrica:
- Query SQL exacta
- Cómo se renderiza (tabla, gráfico, número grande)
- Frecuencia de actualización

Además:
- ¿Input para que el usuario escriba su propia consulta SQL y vea resultados?
- Integración con el NL→SQL del chat
